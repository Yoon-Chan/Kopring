# 1. WebFlux를 이용한 대기열 시스템 프로젝트 With Kotlin Spring

#### 개요

> WebFlux를 이용한 비동기 처리로 대량의 접속 시 대기열 시스템을 이용하여 접근 기술 방법 구현

#### 프로젝트 구축 버전
Spring 3.5.7, Kotlin 1.9.25, java 17

#### 사용 라이브러리
+ thymeleaf
+ Spring MVC
+ webFlux
+ redis reactive
+ Flow, Coroutines
